spectrometer

spectrometer służy do badania składu obiektów

Funkcje:

spectrometer_scan(obj)

Przeskanuj obiekt obj. Zwraca skład obiektu przez Promise.

Polecenia:

"spectrometer scan"

Skanuje obiekt:

"spectrometer scan": {
  target: "id spektrometru",
  material: "id obiektu do przeskanowania"
} -> {
  material: {
    id: "id przetworzonego obiektu",
  },
  composition: skład obiektu
}
