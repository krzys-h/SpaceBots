refinery

refinery służy do przetwarzania obiektów na bazowe surowce, z których są zbudowane. Przetwarzać można zarówno elementy statku jak i asteroidy. Żeby sprawdzić z czego składa się dany obiekt, użyj spektrometru.

Funkcje:

refine(obj)

Przetwórz obiekt obj na bazowe surowce. Jeśli obiekt został przetworzony w całości, znika on. Jeśli obiekt przetworzono tylko częściowo (np. z powodu braku miejsca w magazynie) pozostanie on, ale utraci część ze swoich cech. Zwraca ilość przetworzonego materiału przez Promise.

Polecenia:

"refinery refine"

Przetwarza obiekt:

"refinery refine": {
  target: "id rafinerii",
  store: "id docelowego magazynu",
  material: "id materii do przetworzenia"
} -> {
  id: "id rafinerii",
  refined: ilość przetworzonych materiałów
}
