store

store to magazyn - miejsce gdzie przechowywane są wszystkie zdobyte materiały. Wiele komponentów wymaga wskazania magazynu, który będzie wykorzystywany do operacji

Dodatkowe pola:

"store_capacity"

Pojemność magazynu - maksymalna masa wszystkich zasobów które mogą być zgromadzone w magazynie

"store_stored"

Aktualny stan magazynu - jest to 100-elementowa tablica, gdzie każdy element odpowiada jednemu pierwiastkowi chemicznemu (0=H, 1=He, 2=Li, 3=Be, 4=B, 5=C, itd.)

Funkcje:

store_move(from_store, composition)

Przenosi composition zasobów z from_store do aktywnego magazynu. Jeśli composition nie jest podany, przenoszone są wszystkie zasoby.

Polecenia:

"store move"

Przenosi surowce między magazynami:

"store move": {
  target: "id magazynu docelowego",
  store: "id magazynu źródłowego",
  composition: "zasoby do przeniesienia"
} -> {
  id: "id magazynu docelowego",
  moved: "przeniesione zasoby"
}
